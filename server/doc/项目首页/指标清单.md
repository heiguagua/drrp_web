# 指标清单

-----

## 指标清单列表

### 菜单树
#### /menu
    Type: GET
##### Parameter:
    无参数
##### Result:
    dep_id: 选中的部门名称
    dep_short_name: 部门短名称
    nodes: 子节点数组

### 共享频率、共享级别、空间范围选择框的生成接口
#### /api/sys_dict
    Type: GET
##### Parameter:
    dict_category  '字典类别',
##### Result:
    [id '字典id',  dict_name '字典名称']
    * 数组类型
