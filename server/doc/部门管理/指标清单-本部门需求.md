# 本部门需求指标清单

-----

## 本部门需求指标清单-列表页面

### 列表页面-表格生成接口
#### /api/data_requiement
    Type: GET
##### Parameter:
    quota_name  :  String  '指标名称(顶部右侧指标查询输入框使用)'
    - 默认进入页面时，以上过滤条件可以缺省
##### Result:
    status                          '响应状态',
    create_time                     '创建时间',
    response_time                   '响应时间',
    requiement_name                 '需求名称',
    requiement_desc                 '需求说明',
    share_frequency_name            '共享频率-字典表name',
    share_level_name                '共享级别-字典表name',
    area_range_data_level_name      '分地区数据级别名称'
    linkman                         '联系人',
    contact_phone                   '联系电话',
    dep_name                        '来源部门',
